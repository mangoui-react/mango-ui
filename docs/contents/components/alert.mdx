# Alert

<Description>Displays a description for the user's attention.</Description>

  <AlertDemo />

## Installation

<PackageManagers
  command={{
    npm: 'npm i @melio-ui/alert',
    yarn: 'yarn add @melio-ui/alert',
    pnpm: 'pnpm add @melio-ui/alert',
    bun: 'bun add @melio-ui/alert',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { AlertContent, AlertRoot, AlertStatusIcon } from '@melio-ui/alert';`,
    global: `import { AlertContent, AlertRoot, AlertStatusIcon } from '@melio-ui/react';`,
  }}
/>

- **AlertRoot**: All parts of the alert are included.
- **AlertStatusIcon**: Alert icon changes depending on status.
- **AlertContent**: The content of the alert to be announced by screen readers.

### Another way to import

```js
'use client';

import { Alert } from '@melio-ui/alert';

<Alert.Root>
  <Alert.StatusIcon />
  <Alert.Content>Content1</Alert.Content>
</Alert.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Set the status value

```jsx
<Alert.Root status="success">
  <Alert.StatusIcon />
  <Alert.Content>This is an success alert</Alert.Content>
</Alert.Root>
```
