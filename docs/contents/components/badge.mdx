# Badge

<Description>
  Badges are used to display small numeric values or status descriptors for UI elements.
</Description>

<DemoContainer>
  <BadgeDemo />
  <DemoCodeBlock component="badge" />
</DemoContainer>

## Installation

<PackageManagers
  command={{
    npm: 'npm i @melio-ui/badge',
    yarn: 'yarn add @melio-ui/badge',
    pnpm: 'pnpm add @melio-ui/badge',
    bun: 'bun add @melio-ui/badge',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { BadgeRoot, BadgeContent } from '@melio-ui/badge';`,
    global: `import { BadgeRoot, BadgeContent } from '@melio-ui/react';`,
  }}
/>

- **BadgeRoot**: All parts of the badge are included.
- **BadgeContent**: Elements that display a small numeric value or status description.

### Another way to import

```js
'use client';

import { Badge } from '@melio-ui/badge';

<Badge.Root>
  <Badge.Content />
</Badge.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Setting the maximum value

```jsx
<BadgeRoot max={99}>
  <BadgeContent>{1000}</BadgeContent>
  <UserIcon width="2em" height="2em" />
</BadgeRoot>
```

### Placement

```jsx
<BadgeRoot placement="bottom-right">
  <BadgeContent>{10}</BadgeContent>
  <UserIcon width="2em" height="2em" />
</BadgeRoot>
```

### Only value

```jsx
<BadgeRoot>
  <BadgeContent>{10}</BadgeContent>
</BadgeRoot>
```
