# Dropdown

<Description>
  Displays a menu to the user, offering a series of actions or functions that can be executed by
  clicking buttons.
</Description>

  <DropdownDemo />
  <DemoCodeBlock component="dropdown" />

## Installation

<PackageManagers
  command={{
    npm: 'npm i @mangoui/dropdown',
    yarn: 'yarn add @mangoui/dropdown',
    pnpm: 'pnpm add @mangoui/dropdown',
    bun: 'bun add @mangoui/dropdown',

/>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  DropdownRoot,
  DropdownTrigger,
  DropdownPortal,
  DropdownContent,
  DropdownArrow,
  DropdownItem,
} from '@mangoui/dropdown';`,
    global: `import { 
  DropdownRoot,
  DropdownTrigger,
  DropdownPortal,
  DropdownContent,
  DropdownArrow,
  DropdownItem,
} from '@mangoui/react';`,
  }}
/>

- **DropdownRoot**: All parts of the dropdown are included.
- **DropdownTrigger**: Used to activate or open the Dropdown component.
- **DropdownPortal**: Portal the overlay and content portion into the body.
- **DropdownContent**: Includes the content that will be displayed in the open menu box.
- **DropdownArrow**: An optional arrow element for the dropdown, to be rendered within DropdownContent for visual association.
- **DropdownItem**: It represents a single item in a dropdown menu and can be customized with labels, icons, and actions.

### Another way to import

```js
'use client';

import { Dropdown } from '@mangoui/dropdown';

<Dropdown.Root>
  <Dropdown.Trigger asChild>
    <button type="button">Dropdown</button>
  </Dropdown.Trigger>
  <Dropdown.Portal>
    <Dropdown.Content>
      <Dropdown.Item>DropdownItem - 1</Dropdown.Item>
      <Dropdown.Item>DropdownItem - 2</Dropdown.Item>
      <Dropdown.Item>DropdownItem - 3</Dropdown.Item>
    </Dropdown.Content>
  </Dropdown.Portal>
</Dropdown.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Position of dropdown menu

```jsx
<DropdownRoot>
  <DropdownTrigger asChild>
    <button
      type="button"
      style={{
        border: '1px solid #475569',
        borderRadius: 5,
        padding: 10,
      }}
    >
      Dropdown
    </button>
  </DropdownTrigger>
  <DropdownPortal>
    <DropdownContent side="right" align="start">
      <DropdownItem>DropdownItem - 1</DropdownItem>
      <DropdownItem>DropdownItem - 2</DropdownItem>
      <DropdownItem>DropdownItem - 3</DropdownItem>
    </DropdownContent>
  </DropdownPortal>
</DropdownRoot>
```
