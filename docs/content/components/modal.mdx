# Modal

<Description>
  A modal is a dialog box that overlays the main content to focus the user's attention.
</Description>

  <ModalDemo />

## Import

<ImportSyntax
  command={{
    individual: `import { 
  ModalRoot,
  ModalTrigger,
  ModalPortal,
  ModalBackdrop,
  ModalContent,
  ModalClose,
} from '@melio-ui/modal';`,
    global: `import { 
  ModalRoot,
  ModalTrigger,
  ModalPortal,
  ModalBackdrop,
  ModalContent,
  ModalClose,
} from '@melio-ui/react';`,
  }}
/>

- **ModalRoot**: All parts of the modal are included.
- **ModalTrigger**: Used to activate or open the Modal component.
- **ModalPortal**: Portal the overlay and content portion into the body.
- **ModalBackdrop**: A layer that covers the inactive portion of the view when a dialog box is opened.
- **ModalContent**: Includes the content that will be displayed in the open dialog box.
- **ModalClose**: The button that closes the dialog.

### Another way to import

```js
'use client';

import { Modal } from '@melio-ui/modal';

<Modal.Root>
  <Modal.Trigger asChild>
    <button type="button">Open Modal</button>
  </Modal.Trigger>
  <Modal.Portal>
    <Modal.Backdrop />
    <Modal.Content>
      <Modal.Close asChild>
        <button type="button" />
      </Modal.Close>
      <div>Modal Content</div>
    </Modal.Content>
  </Modal.Portal>
</Modal.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Position

```jsx
<ModalRoot>
  <ModalTrigger asChild>
    <button
      type="button"
      style={{
        border: '1px solid #2e68da',
        padding: 5,
        borderRadius: 5,
        backgroundColor: '#2e539b',
      }}
    >
      Open Modal
    </button>
  </ModalTrigger>
  <ModalPortal>
    <ModalBackdrop />
    <ModalContent style={{ top: 200 }}>
      <ModalClose asChild>
        <button type="button">
          <svg
            width="1em"
            height="1em"
            viewBox="0 0 24 24"
            fill="currentColor"
            xmlns="http://www.w3.org/2000/svg"
          >
            <path
              d="M19 6.41L17.59 5 12 10.59 6.41 5 5 6.41 10.59 12 5 17.59 6.41 19 12 13.41 17.59 19 19 17.59 13.41 12z"
              fill="currentColor"
            />
          </svg>
        </button>
      </ModalClose>
      <div>Modal Content</div>
      <div>
        <ModalClose asChild>
          <button
            type="button"
            style={{
              color: '#208aff',
              backgroundColor: 'transparent',
              marginRight: 3,
            }}
          >
            Close
          </button>
        </ModalClose>
      </div>
    </ModalContent>
  </ModalPortal>
</ModalRoot>
```
