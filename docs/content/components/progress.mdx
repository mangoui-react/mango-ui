# Progress

<Description>Displays a progress bar indicating task completion.</Description>

  <ProgressDemo />
  <DemoCodeBlock component="progress" />
</DemoContainer>

## Installation

<PackageManagers
  command={{
    npm: 'npm i @mangoui/progress',
    yarn: 'yarn add @mangoui/progress',
    pnpm: 'pnpm add @mangoui/progress',
    bun: 'bun add @mangoui/progress',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  ProgressRoot,
  ProgressIndicator,
} from '@mangoui/progress';`,
    global: `import { 
  ProgressRoot,
  ProgressIndicator,
} from '@mangoui/react';`,
  }}
/>

- **ProgressRoot**: All parts of the progress are included.
- **ProgressIndicator**: Render the progress indicator. You can style it yourself.

### Another way to import

```js
'use client';

import { Progress } from '@mangoui/progress';

<Progress.Root value={30}>
  <Progress.Indicator />
</Progress.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Indeterminate

```jsx
<ProgressRoot indeterminate>
  <ProgressIndicator />
</ProgressRoot>
```

### Custom Indicator

```jsx
<ProgressRoot value={30}>
  <ProgressIndicator style={{ backgroundColor: '#ff4d4f' }} />
</ProgressRoot>
```

### Max

```jsx
<ProgressRoot value={100} max={200}>
  <ProgressIndicator />
</ProgressRoot>
```
