# Slider

<Description>An input that allows the user to choose a value within a specified range.</Description>

<DemoContainer>
  <SliderDemo />
  <DemoCodeBlock component="slider" />
</DemoContainer>

## Installation

<PackageManagers
  command={{
    npm: 'npm i @mangoui/slider',
    yarn: 'yarn add @mangoui/slider',
    pnpm: 'pnpm add @mangoui/slider',
    bun: 'bun add @mangoui/slider',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  SliderRoot,
  SliderTrack,
  SliderRange,
  SliderThumb,
  SliderThumbLabel,
} from '@mangoui/slider';`,
    global: `import { 
  SliderRoot,
  SliderTrack,
  SliderRange,
  SliderThumb,
  SliderThumbLabel,
} from '@mangoui/react';`,
  }}
/>

- **SliderRoot**: All parts of the slider are included.
- **SliderTrack**: The unfilled portion of the slider that displays the track.
- **SliderRange**: The filled portion of the slider.
- **SliderThumb**: The knob used to adjust the slider value.
- **SliderThumbLabel**: The label displayed for slider values.

### Another way to import

```js
'use client';

import { Slider } from '@mangoui/slider';

<Slider.Root defaultValue={20}>
  <Slider.Track>
    <Slider.Range />
  </Slider.Track>
  <Slider.Thumb>
    <Slider.ThumbLabel />
  </Slider.Thumb>
</Slider.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Disabled

```jsx
<SliderRoot defaultValue={30} disabled>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb />
</SliderRoot>
```

### Set the minimum and maximum values

```jsx
<SliderRoot min={10} max={120} defaultValue={30}>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb />
</SliderRoot>
```

### Set step size

```jsx
<SliderRoot min={10} max={120} step={10} defaultValue={30}>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb />
</SliderRoot>
```

### Display Thumb Label

```jsx
<SliderRoot defaultValue={20}>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb>
    <Slider.ThumbLabel />
  </SliderThumb>
</SliderRoot>
```

### Orientation

```jsx
<SliderRoot orientation="vertical" style={{ height: 150 }}>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb>
    <Slider.ThumbLabel />
  </SliderThumb>
</SliderRoot>
```

### Range

```jsx
<SliderRoot defaultValue={[20, 60]}>
  <SliderTrack>
    <SliderRange />
  </SliderTrack>
  <SliderThumb />
  <SliderThumb />
</SliderRoot>
```
