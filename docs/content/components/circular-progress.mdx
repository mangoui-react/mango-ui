# CircularProgress

<Description>
  Circular progression is used to indicate the progress of both deterministic and indeterminate
  processes.
</Description>

  <CircularProgressDemo />
  <DemoCodeBlock component="circular-progress" />

## Installation

<PackageManagers
  command={{
    npm: 'npm i @melio-ui/circular-progress',
    yarn: 'yarn add @melio-ui/circular-progress',
    pnpm: 'pnpm add @melio-ui/circular-progress',
    bun: 'bun add @melio-ui/circular-progress',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  CircularProgressRoot,
  CircularProgressIndicator,
  CircularProgressLabel,
} from '@melio-ui/circular-progress';`,
    global: `import { 
  CircularProgressRoot,
  CircularProgressIndicator,
  CircularProgressLabel,
} from '@melio-ui/react';`,
  }}
/>

- **CircularProgressRoot**: All parts of the circular-progress are included.
- **CircularProgressIndicator**: Render the progress indicator. You can style it yourself.
- **CircularProgressLabel**: Displays progress as text.

### Another way to import

```js
'use client';

import { CircularProgress } from '@melio-ui/circular-progress';

<CircularProgress.Root>
  <CircularProgress.Indicator />
  <CircularProgress.Label>30%</CircularProgress.Label>
</CircularProgress.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Indeterminate

```jsx
<CircularProgressRoot indeterminate>
  <CircularProgressIndicator />
</CircularProgressRoot>
```

### Custom Indicator

```jsx
<CircularProgressRoot value={60}>
  <CircularProgressIndicator
    strokeWidth={15}
    bgStyle={{ stroke: '#d9d9d9' }}
    barStyle={{ stroke: '#ff4d4f' }}
  />
</CircularProgressRoot>
```

### Max

```jsx
<CircularProgressRoot value={80} max={200}>
  <CircularProgressIndicator />
  <CircularProgressLabel />
</CircularProgressRoot>
```
