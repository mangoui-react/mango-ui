# Tooltip

<Description>
  A popup displaying information when an element receives keyboard focus or mouse hover.
</Description>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  TooltipRoot,
  TooltipTrigger,
  TooltipPortal,
  TooltipContent,
  TooltipArrow,
} from '@melio-ui/tooltip';`,
    global: `import { 
  TooltipRoot,
  TooltipTrigger,
  TooltipPortal,
  TooltipContent,
  TooltipArrow,
} from '@melio-ui/react';`,
  }}
/>

- **TooltipRoot**: All parts of the tooltip are included.
- **TooltipTrigger**: Used to activate or open the Tooltip component.
- **TooltipPortal**: Portal the overlay and content portion into the body.
- **TooltipContent**: Includes the content that will be displayed in the open tooltip.
- **TooltipArrow**: An optional arrow element for the tooltip, to be rendered within TooltipContent for visual association.

### Another way to import

```js
'use client';

import { Tooltip } from '@melio-ui/tooltip';

<Tooltip.Root>
  <Tooltip.Trigger asChild>
    <button>Tooltip</button>
  </Tooltip.Trigger>
  <Tooltip.Portal>
    <Tooltip.Content>
      tooltip message
      <Tooltip.Arrow />
    </Tooltip.Content>
  </Tooltip.Portal>
</Tooltip.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Display arrow

```jsx
<TooltipRoot>
  <TooltipTrigger asChild>
    <button type="button" style={{ width: 200, border: '1px solid #5f76bb', padding: 5 }}>
      Hover me
    </button>
  </TooltipTrigger>
  <TooltipPortal>
    <TooltipContent>
      tooltip message
      <TooltipArrow />
    </TooltipContent>
  </TooltipPortal>
</TooltipRoot>
```

### Position Settings

```jsx
<TooltipRoot>
  <TooltipTrigger asChild>
    <button type="button" style={{ width: 200, border: '1px solid #5f76bb', padding: 5 }}>
      Hover me
    </button>
  </TooltipTrigger>
  <TooltipPortal>
    <TooltipContent side="bottom" align="end">
      tooltip message
      <TooltipArrow />
    </TooltipContent>
  </TooltipPortal>
</TooltipRoot>
```
