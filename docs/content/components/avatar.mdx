## Import

<ImportSyntax
  command={{
    individual: `import { 
  AvatarRoot,
  AvatarImg,
  AvatarGroup,
  AvatarGroupContent,
  AvatarExtra,
} from '@melio-ui/avatar';`,
    global: `import { 
  AvatarRoot,
  AvatarImg,
  AvatarGroup,
  AvatarGroupContent,
  AvatarExtra,
} from '@melio-ui/react';`,
  }}
/>

- **AvatarRoot**: All parts of the avatar are included.
- **AvatarImg**: The image to render. By default, it will only be rendered when loaded.
- **AvatarGroup**: It is a wrapper that stacks multiple avatars together.
- **AvatarGroupContent**: When using AvatarGroup, an avatar component is included.
- **AvatarExtra**: When using AvatarGroup, extra space above the maximum number of avatars is displayed.

### Another way to import

```js
'use client';

import { Avatar } from '@melio-ui/avatar';

<Avatar.Root>
  <Avatar.Img />
</Avatar.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### Loading the image.

```jsx
<AvatarRoot>
  <AvatarImg src="http://bit.ly/3XZx3ZU" />
</AvatarRoot>
```

### Grouping

```jsx
<AvatarGroup>
  <AvatarGroupContent>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
  </AvatarGroupContent>
</AvatarGroup>
```

### Setting the maximum value

```jsx
<AvatarGroup max={2}>
  <AvatarGroupContent>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
    <AvatarRoot>
      <UserIcon />
    </AvatarRoot>
  </AvatarGroupContent>
  <AvatarExtra />
</AvatarGroup>
```
