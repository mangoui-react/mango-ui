# Switch

<Description>A control that lets the user toggle between checked and unchecked states.</Description>

  <SwitchDemo />
  <DemoCodeBlock component="switch" />
</DemoContainer>

## Installation

<PackageManagers
  command={{
    npm: 'npm i @mangoui/switch',
    yarn: 'yarn add @mangoui/switch',
    pnpm: 'pnpm add @mangoui/switch',
    bun: 'bun add @mangoui/switch',
  }}
/>

## Import

<ImportSyntax
  command={{
    individual: `import { 
  SwitchRoot,
  SwitchTrack,
  SwitchThumb,
  SwitchLabel,
} from '@mangoui/switch';`,
    global: `import { 
  SwitchRoot,
  SwitchTrack,
  SwitchThumb,
  SwitchLabel,
} from '@mangoui/react';`,
  }}
/>

- **SwitchRoot**: All parts of the switch are included.
- **SwitchTrack**: It represents the switch's background, indicating on and off states with different colors or styles.
- **SwitchThumb**: It is the interactive element that toggles between on and off states.
- **SwitchLabel**: It displays text or content next to the switch to clarify its purpose.

### Another way to import

```js
'use client';

import { Switch } from '@mangoui/switch';

<Switch.Root>
  <Switch.Track>
    <Switch.Thumb />
  </Switch.Track>
  <Switch.Label>Label</Switch.Label>
</Switch.Root>;
```

> 'use client' must be used when rendering on the server side.

## Examples

### defaultChecked

```jsx
<SwitchRoot defaultChecked>
  <SwitchTrack>
    <SwitchThumb />
  </SwitchTrack>
</SwitchRoot>
```

### disabled

```jsx
<SwitchRoot disabled>
  <SwitchTrack>
    <SwitchThumb />
  </SwitchTrack>
</SwitchRoot>
```

### Display Label

```jsx
<SwitchRoot defaultChecked>
  <SwitchTrack>
    <SwitchThumb />
  </SwitchTrack>
  <SwitchLabel>Label</SwitchLabel>
</SwitchRoot>
```

### Display checked/unchecked status text within the SwitchTrack.

```jsx render=true
function DisplayText() {
  const [checked, setChecked] = React.useState(false);

  return (
    <SwitchRoot checked={checked} onCheckedChange={setChecked}>
      <SwitchTrack>
        <SwitchThumb />
        {checked ? (
          <span style={{ margin: '0 28px 0 5px', color: '#fff' }}>open</span>
        ) : (
          <span style={{ margin: '0 5px 0 28px' }}>close</span>
        )}
      </SwitchTrack>
    </SwitchRoot>
  );
}

render(<DisplayText />);
```
